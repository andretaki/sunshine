# Premium Fonts Directory

This directory contains the premium brand fonts for The Sunshine Effect.

## Required Fonts (To Be Purchased)

### 1. Belvare
- **Usage**: Display headings (H1, H3), logo lockups
- **Weight**: Bold (700)
- **Format**: .woff2
- **File**: `Belvare-Bold.woff2`
- **Purchase**: https://www.myfonts.com/collections/belvare-font-latinotype

### 2. Laro Soft Bold
- **Usage**: Subheadings (H2, H4), taglines, short emphasis text
- **Weight**: Bold (700)
- **Format**: .woff2
- **File**: `LaroSoft-Bold.woff2`
- **Purchase**: https://www.myfonts.com/collections/laro-font-latinotype

## Current Status
⚠️ **Fonts not yet purchased** - Currently using fallback fonts:
- Belvare → Times New Roman, serif
- Laro Soft → system-ui, sans-serif

## Installation
Once purchased, place the .woff2 files in this directory and the site will automatically use them.
